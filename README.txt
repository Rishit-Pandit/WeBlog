To run the Development Server:
	1. Right-Click on the "server.bat" file.
	2. click on 'Run as Administrator.'
	3. When you see these lines open your browser : 
		"System check identified no issues (0 silenced)."
		"Django version 3.0.4, using settings 'WeBlog.settings'"
		"Starting development server at http://127.0.0.1:4000/"
		"Quit the server with CTRL-BREAK."
	4. Do not close the file untill you need to stop the Development Server!
	4. Type 127.0.0.1:4000/ in your browser's address bar and hit Enter.
	5. After a few seconds the website would we fully loaded(If not refresh the page again.)

To stop the Development Server:
	1. Goto the 'server.bat' file and hit "Ctrl + c"
	2. Again hit "Ctrl + c"
	3. The Development Server will stop and the file will close automatically.
